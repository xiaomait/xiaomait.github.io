🎁 关于博客

在初中的时候就很想搭建一个自己的博客，刚好在某论坛，无意发现了 WordPress 的搭建教程，其中那教程还包含了免费的虚拟主机，我突然兴致就起来了，搭建好之后，非常炫耀的展示给同学看，但是访问的速度很不理想 🤣 十几个人就卡死了（那时候还不知道高并发），不知道是 WordPress 太重量了，还是虚拟主机配置太低了，那时候还不懂，现在也忘了那虚拟主机的配置，记得那时候下课也不忘掏出手机来看看自己搭建的成果（那时候严谨带手机的，不要学哦）然后就快中考了，我也渐渐的放下了博客把心思放在学业上，直到 2020 年的 1 月 14 日（情人节），时隔多年我突然又想搭建一个个人博客，但是服务器的昂贵这使我可能坚持不了多久，最终在 2021 年的 11 月 11 日（双十一）它倒闭了…..我在想也没有免费轻量级的博客呢？所以找了度娘，发现了我的答案，那便是 Hexo ➕ Github，最终花了3天时间的研究琢磨后搭建而成，并且搭载了强大的 Butterfly 主题。

So(所以)

本博客使用 Hexo ➕ GitHub 搭建而成。

主题使用的是一款简单的卡片式 UI 设计主题 Butterfly

使用 jsDelivr 进行静态资源加速！
