超星学习通小助手安装教程

发表于2022-03-02|更新于2022-03-20|学习教程
|阅读量:

前言

很多小伙伴由于受到学习通（堵）的摧残与折磨，纷纷去寻找插件和脚本来一键完成任务
但是很多小伙伴由于是初次接触浏览器扩展（也叫插件），所以在安装插件方面还有不少困惑，首先你需要知道插件的扩展名为 .crx 或 .xpi
而Tampermonkey是一款免费的浏览器扩展和最为流行的用户脚本管理器，它适用于 Chrome, Microsoft Edge, Safari, Opera Next, 和 Firefox等众多浏览器。
今天给大家带来利用 Microsoft Edge 浏览器（windows10以上电脑自带）安装 Tampermonkey 和学习通助手脚本。
注意：最近超星一直更新，不保证以后100%有效，本页持续更新最新脚本。

教程

1、安装 Tampermonkey

打开 Microsoft Edge 浏览器（windows10以上电脑自带）

点击下载 Tampermonkey，它会自动将 Tampermonkey 安装好,这时你就会发现浏览器右上去出现新安装好的插件。

点击下载 Tampermonkey

因为是国外的地址所以打开比较缓慢请耐心等待，加载完成之后会自动弹出以下界面则表示安装成功（特殊情况除外）。

2、安装 学习通助手

一切安装好之后就可以安装学习通助手脚本了，这里有很多脚本都是以前的，自2021.10学习通更新之后，增加了屏蔽高倍数播放代码，很多人都暂停更新脚本了，这里我找到一个依旧可用的插件

点我下载 学习通助手

点击下载进去安装之后，我们就可以前往学习通，随便打开一个科目的章节，自己配置之后即可开启暴走（挂科) 模式,倍数播放建议不要太高，最大为16倍数。

最后特别提醒：秒过有风险，倍数需谨慎。
