以下教程仅用于个人研究、学习代码目的，为各位同学节省宝贵的学习时间，请勿用于商业用途。

随着开学季到来，大批大学生又要进入到各种大小网课当中，每日的刷课，刷作业搞的头都大了，那么今天给大家带来自动刷课答题脚本安装教程。

脚本猫的安装

脚本猫 ,一个可以执行用户脚本的浏览器扩展,万物皆可脚本化,让你的浏览器可以做更多的事情!

持续兼容油猴脚本中,已兼容 90%+的油猴脚本,更多油猴特性完善中,并且另外支持更加强大的后台脚本和定时脚本!

提示

请务必选择对应的浏览器安装扩展，否则无法正常安装成功。
windows 用户推荐使用微软自带的Microsoft Edge 浏览器

Mac 用户推荐使用Firefox 浏览器

Google Chrome 浏览器商城需要科学上网工具 🔧 才可以正常访问

其他浏览器，可以从Github Release 页面中,下载zip/crx文件手动进行安装,但是这是不推荐的，因为它操作难度相对较大，且无法自动更新扩展。

推荐从下列扩展商店中安装扩展

  • Microsoft Edge
  • Google Chrome
  • Firefox

安装好后，扩展程序中就拥有ScriptCat了

脚本的安装

OCS 网课助手

这里提供 OCS 网课助手，支持各平台网课学习，支持超星学习通，知道智慧树，智慧职教（职教云），等网课的学习，作业，考试功能。

官网/教程： https://docs.ocsjs.com/

所有功能请看： - OCS 网课助手 - 功能列表

点击这里安装

安装好后不会有任何提示，你可以直接访问超星学习通或者其他网课平台就可以看到它已经激活了.

全平台自动答题助手

OCS 的答题准确率实在不敢恭维，所以这里推荐自动答题助手 内置免费题库和收费题库

  • 免费题库：涵盖大部分公选课答案
  • 收费题库：涵盖面积更广，并且没有搜索到的题库会自动加入后台，实时采集答案

它的答题是非常强大的，支持【超星学习通】【智慧树】【职教云系列】【雨课堂】【继续教育类】【小鹅通】【安徽继续教育】 【上海开放大学】 【华侨大学自考网络助学平台】【人卫慕课】【国家开放大学】【浙江省高等学校在线开放课程共享平台】【国地质大学远程与继续教育学院】【浙江省高等教育自学考试网络助学平台】 【湖南高等学历继续教育】 【优学院】 【学起 Plus】【青书学堂】 【学堂在线】【英华学堂】【广开网络教学平台】等平台的测验考试，内置题库，自动答题功能全聚合。

点击这里安装

题海 x 划词搜题

全平台自带答题助手分为免费题库和付费题库，有时候免费题库就会搜不到题，对于白嫖党来说,这里就推荐使用这个脚本了。

所有网页均支持划词搜题,也可输入文本搜题悬浮窗可拖动可关闭,可个性化设置解除网页禁止复制限制适用于各类问答,网课问题,竞赛问题,专业术语,业务名称,情景问题,在线作业等。

点击这里安装

马小智公众号搜题

这是由本人经营的公众号，由河北瓜子科技有限公司提供部分技术接口.
在这里也提供了搜题功能，同时支持图片语音搜题功能噢！
